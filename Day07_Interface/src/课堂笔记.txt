六，static关键字
	翻译成中文就是：静态的
	静态的属性和方法，是类的
	也就是static修饰的属性，或者方法
	是类的
	
	！！！！！不是依赖于对象的

	1，静态方法中不能使用非静态变量
	2，静态的方法或属性，可以理解为是共享的状态
		每个对象操作的都是一个相同的属性或方法。
		注意是相同的，不是两个东西一摸一样的相同，
		指的是在内存中，就存在一个！！！！
	3，静态的属性和方法，可以使用类名直接调用
		eg：
			static int a;
			static void show(){}
			
		类名.a;
		类名.show();
